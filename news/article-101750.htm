anaconda下载安装

注：在Anaconda安装的过程中，比较容易出错的环节是环境变量的配置，所以大家在配置环境变量的时候，要细心一些。

  • 步骤一：输入链接：https://www.anaconda.com 登录Anaconda官网。
  • 步骤二：鼠标选中“Products”，点击“Indiviaual Edition”选项（Individual Edition是免费版的）。
  • 步骤三：根据系统选择对应版本下载，windows 选择Windows版本“64-Bit Graphical Installer
  • 步骤四：双击已下载的exe文件，进行安装。
  • 步骤五：点击“Next”。
  • 步骤六：点击“I Agree”。
  • 步骤七：选择“Just Me”，点击“Next”。（若电脑有多个用户，则选择“All Users”）

    步骤八：设置Anaconda的安装路径，路径名称最好为全英文，随后点击“Next”。

  • 步骤九：点击“Install”。一般选择自动将Anaconda添加到系统路径。这里就不选择自动将Anaconda添加到系统路径了（因为我之前安装Anaconda的时候都是手动添加路径，使用的时候没有出现过问题，所以本次安装我仍然手动配置路径，我没有试过自动添加路径，所以不知道会出什么问题，大家感兴趣可以尝试。）
  • 步骤十：点击“Next”。
  • 步骤十一：点击“Next”。
    <b
  • 步骤十二：点击“Finish”。

进行 pip install。

这里选择 https://www.lfd.uci.edu/~gohlke/pythonlibs/，因为所有链接都在一个页面，搜索比较方便。它由荧光动力学实验室维护，隶属于美国国立卫生研究院加利福尼亚大学生物医学荧光光谱研究中心。

whl 离线安装

下载完成后，进行安装，安装步骤如下：

  • （1）使用 win+R 打开命令窗口
  • （2）输入命令 cd /d D:****\Anaconda\envs\py36\Scripts 打开 whl 文件所在文件夹
  • （3） 输入命令 pip install ***.whl 就可以实现离线安装啦

tar 离线安装

有一些三方库没有 whl 文件，只有 .tar.gz 文件；或者没有你使用平台和python对应版本的whl 文件，怎么办呢？

  • 1、下载.tar.gz文件，解压
  • 2、从命令行界面（cmd）进入上述解压目录
  • 3、使用命令：python setup.py install 进行安装

一、python安装/卸载第三方包

（1）win+R输入cmd，打开命令窗口
（2）安装第三方包：安装指令 pip install xxx (xxx，需安装的包名)
（3）卸载第三方包：卸载指令 pip uninstall xxx (xxx，需卸载的包名)

二、查看python版本号

（1）win+R输入cmd，打开命令窗口
（2）输入指令：python --version

输入指令后，可看到小编安装的python版本是：python 3.8.5

三、查看所有第三方包

（1）win+R输入cmd，打开命令窗口
（2）输入指令：pip list

四、查看某第三包信息、版本号

（1）win+R输入cmd，打开命令窗口
（2）输入指令：pip show xxx（xxx，需查看的包名）

  • 4、例如我的python3.8的残留文件是 python.exe 、 python38.dll，在打开的目录下 del 这两个文件：del python.exe , del python38.dll
